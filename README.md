# NEP based English Learning Guide for Grades 3 to 5

An interactive learning platform aligned with India's National Education Policy 2020.

## Live Site

https://bksourabh.github.io/india_education/

## Tech Stack

- Vite + React 18 + TypeScript
- Tailwind CSS v3
- React Router v6 (HashRouter)
- Framer Motion
- GitHub Actions for auto-deployment

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Deployment happens automatically via GitHub Actions on every push to `main`.
